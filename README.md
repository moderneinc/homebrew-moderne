# homebrew-moderne
Homebrew tap for Moderne

## Usage

### Mod CLI
#### Stable
```shell
brew install moderneinc/moderne/mod
```
#### Staging
```shell
brew install moderneinc/moderne/mod --head
```

### Moddy Desktop
#### Stable
```shell
brew install --cask moderneinc/moderne/moddy-desktop
```
#### Prerelease
```shell
brew install --cask moddy-desktop-prerelease
```
